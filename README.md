# Streamlit_app

### Описание приложения

1. на главной странице показаны данные котировок компании Apple с 30.09.2010 по 30.09.2024
2. на первой странице реализовано исследование чаевых кафе
- скачайте файл [tips](https://raw.githubusercontent.com/mwaskom/seaborn-data/master/tips.csv)
- на боковой панели имеется виджет кнопки загрузки файлов, нажмите ее и загрузите ранее скаченный файл **tips**
- при нажатии кнопок типа **Построить**, **Нарисовать** будет создан график
- на боковой панели имеется виджет кнопки загрузки с помощью нее можно скачать текущий график
